// Common Valid Usage
// Common to indirect drawing commands with a pname:countAddressRange parameter
// {drawCount} is the draw count or max draw count of the command
// {indirectStruct} is the name of the structure containing indirect parameters
  * [[VUID-{refpage}-pInfo-13110]]
    If pname:pInfo->{drawCount} is greater than `0`,
    pname:pInfo->addressRange.size must: be greater than or equal to
    [eq]#(pname:pInfo->{drawCount} - 1) {times}
    pname:pInfo->addressRange.stride {plus}
    code:sizeof(slink:{indirectStruct})#
  * [[VUID-{refpage}-pInfo-13111]]
    If pname:pInfo->{drawCount} is greater than 1,
    pname:pInfo->addressRange.stride must: not be 0
  * [[VUID-{refpage}-pInfo-13112]]
    If pname:pInfo->addressRange.stride is not 0,
    pname:pInfo->addressRange.stride must: be greater than or equal to
    code:sizeof(slink:{indirectStruct})
  * [[VUID-{refpage}-pInfo-13113]]
    If pname:pInfo->addressRange.stride is not 0,
    pname:pInfo->addressRange.stride must: be a multiple of `4`
// Common Valid Usage
