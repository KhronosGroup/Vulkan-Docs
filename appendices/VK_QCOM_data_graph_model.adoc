=== Description

This extension supports new
elink:VkPhysicalDeviceDataGraphProcessingEngineTypeARM, and
elink:VkPhysicalDeviceDataGraphOperationTypeARM types for data graph
pipelines added in `apiext:VK_ARM_data_graph`.

A new pipeline cache type is also added to seamlessly import ML models such
as ONNX through QNN workflow, and run them on the device or an external
compute engine.
