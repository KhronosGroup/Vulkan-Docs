=== Description

This extension adds the equivalent of
ename:VK_IMAGE_CREATE_MULTISAMPLED_RENDER_TO_SINGLE_SAMPLED_BIT_EXT to
swapchain create flags such that the functionality of the
`apiext:VK_EXT_multisampled_render_to_single_sampled` extension can be used
with its images.
