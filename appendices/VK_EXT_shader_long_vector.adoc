=== Description

This extension adds support for using vector types with more than four
components in SPIR-V.

Long vector types are defined by the
{spirv}/EXT/SPV_EXT_long_vector.html[`SPV_EXT_long_vector`] SPIR-V extension
and can be used with the
{GLSLregistry}/ext/GLSL_EXT_long_vector.txt[`GL_EXT_long_vector`] GLSL
extension.
